**பொருள்:**  
சிறுத்துவிடாமலும், தலைவனை வெறுத்து விடாமலும், பயன்படாத நிலை இல்லாமலும் உள்ள படைதான் வெற்றி பெற முடியும்.

**Meaning (English):**  
An army can triumph (over its foes) if it is free from diminution; irremediable aversion and poverty.
