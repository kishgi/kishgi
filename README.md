காதல் பெருவெள்ளமானது நாணம், நல்ல ஆண்மை எனப்படும் தோணிகளை அடித்துக்கொண்டு போய்விடும் வலிமை வாய்ந்தது.

**Meaning (English):**  
The raft of modesty and manliness, is, alas, carried-of by the strong current of lust.
