**பொருள்:**  
எந்தவொரு காலகட்டத்திலும் தன்னைத்தானே உயர்வாக எண்ணிடும் தற்பெருமைகொண்டு நன்மை தராத செயல்களில் ஈ.டுபடக் கூடாது.

**Meaning (English):**  
Let no (one) praise himself, at any time; let him not desire to do useless things.
