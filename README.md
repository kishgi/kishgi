**பொருள்:**  
ஒரு செயலில் ஈ.டுபடுவதற்கு ஏற்றவனா என்பதை ஆராய்ந்து அறிந்த பிறகே, அவனை அந்தச் செயலில் ஈ.டுபடுத்த வேண்டும்.

**Meaning (English):**  
Having considered what work a man is fit for, let (the king) employ him in that work.
