**பொருள்:**  
காதல் நோயை என்னால் மறைக்கவும் முடியவில்லை; இதற்குக் காரணமான காதலரிடம் நாணத்தால் உரைக்கவும் முடியவில்லை.

**Meaning (English):**  
I cannot conceal this pain, nor can I relate it without shame to him who has caused it.
